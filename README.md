# saikirankonda.com — Personal Portfolio

My portfolio site — a cinematic video hero plus About, Projects, and Contact sections.

- Three.js and GSAP for motion
- CSS Modules
- Deployed on Vercel

## Run locally
\`\`\`bash
npm install
npm run dev
\`\`\`
Open http://localhost:3000

## Structure
- `app/` — pages and layout
- `components/` — VideoIntro (hero), About, Projects, Contact
- `public/intro.mp4` — hero video
